# MSI 2026 - Price Mapping

Interactive price mapping visualization for MSI 2026 teams.

## Features

- **Elo → $ Curves**: Compare current logistic pricing vs proposed exponential pricing
- **Sensitivity Analysis**: % price change per Elo point
- **Shock Test**: Visualize price impact of Elo shocks across teams
- **Match Simulator**: Simulate matches and see price impacts

## Live Demo

https://igorai.github.io/msi-2026-price-mapping

## Tech Stack

- React + Vite
- Recharts for visualizations
- GitHub Pages for hosting
